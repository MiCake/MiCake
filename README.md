## ✨ 特性

- 快速将您的项目转换为DDD风格。
- 实现了DDD战术模式提出的所有部件。
- 提供项目常用的基础功能（全局错误捕获，对象映射等）。
- 通过模块进行解耦，您可以根据自身需求使用特定的MiCake模块。
