## ✨ 特性

- 提炼自企业级中后台产品的交互语言和视觉风格。
- 开箱即用的高质量 Razor 组件，可在多种托管方式共享。
- 支持基于 WebAssembly 的客户端和基于 SignalR 的服务端UI事件交互。
- 支持渐进式 Web 应用（PWA）
- 使用 C# 构建，多范式静态语言带来高效的开发体验。
- 基于 .NET Standard 2.1，可直接引用丰富的 .NET 类库。
- 可与已有的 ASP.NET Core MVC、Razor Pages 项目无缝集成。
